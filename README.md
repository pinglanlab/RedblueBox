# RedblueBox
1、背景  
红蓝工具箱是由凭阑实验室安全团队开发的一款面向Windows系统的图形化（GUI）安全渗透集成工具箱。它旨在解决安全研究人员和渗透测试工程师在日常工作中需要单独下载、配置多种工具及其复杂运行环境的痛点。  
2、概述  
红蓝工具箱集成了超过百款工具，覆盖了渗透测试的多个关键环节，主要可以分为以下几大类：  
1）WebShell管理工具  
集成了多款主流的WebShell管理客户端，如哥斯拉（Godzilla）、冰蝎（Behinder）、中国蚁剑（AntSword）、天蝎权限管理工具等，方便对已获取权限的服务器进行管理和操作。  
2）信息收集与探测工具  
提供了全面的信息收集解决方案，包括子域名探测（OneForAll）、敏感信息探测（webpack、APP信息收集）、指纹识别（Tide、Ehole）、目录扫描（DirSearch）、网络空间测绘（FOFA Search）以及企业资产收集（EnScan）等。  
3）渗透利器与漏洞扫描  
包含了渗透测试中常用的核心工具，例如BurpSuite、CobaltStrike、SQLMap、Xray、Goby、Fscan、Nuclei等，用于漏洞扫描、流量代理、SQL注入、内网渗透等。  
4）综合漏洞探测与利用工具  
针对特定框架和组件的漏洞，提供了专门的探测和利用工具，覆盖了如Spring、Thinkphp、Weblogic、OA系统（如通达、泛微）、Fastjson、Redis、Nacos等多种常见组件和框架。  
5）其他实用工具  
内网渗透： 包含内网横向移动、AD域评估、Mimikatz、PotatoTool等工具。  
代理工具： 集成了Fiddler、Proxifier等代理工具的汉化版。  
加密解密： 提供多种编码解码和加密解密小工具。  
提权工具： 如winPEASany等。  
3、优势  
1）开箱即用，环境免配  
最大的优势在于将所有工具及其依赖环境预先配置好，用户无需再为配置Python、Java等环境而烦恼，极大地简化了准备工作。  
2）高度集成，功能全面  
一个工具箱覆盖了从信息收集、漏洞探测、利用到权限维持的完整渗透测试流程，避免了在不同工具间频繁切换的麻烦。  
3）界面简洁，易于操作  
提供图形化用户界面，布局清晰，并支持多种皮肤UI热切换，操作直观流畅。  
4）持续更新，生态活跃  
开发团队持续维护更新，不仅会升级内置工具的版本，还会根据安全形势新增实用工具，并修复已知问题，保证了工具箱的时效性和稳定性。  
5）开源透明，支持二开  
工具箱本体采用Python开发且代码开源，个人用户可以自行审查代码，也可以根据个人需求进行二次开发和功能扩展。  
4、价值  
1）提升工作效率  
对于安全从业者而言，工具箱将繁琐的工具准备过程简化为“一键启动”，显著提升了渗透测试和应急响应工作的效率。  
2）降低学习门槛  
对于网络安全初学者，工具箱提供了一个绝佳的学习平台。通过它，初学者可以快速接触到业界主流的安全工具，并了解它们在实际工作流中的应用场景。  
3）标准化作业流程  
将常用工具标准化地集成在一起，有助于团队内部形成统一的作业流程和工具链，便于协作和知识共享。
